# Mi Plan FIRE · Fábrica de calculadoras

La **puerta de entrada SEO** a la app. Diez calculadoras estáticas, rápidas e indexables que atacan las queries que hoy ganan tus competidores y **embudan cada visita hacia `app.miplanfire.com`**.

Todas salen de **un solo config**. Añadir una nueva = editar datos, no escribir una página.

## Arranque rápido

```bash
npm run build      # genera /dist
npm run preview    # míralo en http://localhost:4321
```

## Las 10 calculadoras

| # | Slug | Query objetivo |
|---|------|----------------|
| 1 | `regla-del-4-por-ciento` | regla del 4%, número FIRE |
| 2 | `cuanto-necesito-para-ser-libre` | cuánto necesito para ser libre |
| 3 | `interes-compuesto` | calculadora interés compuesto |
| 4 | `coast-fire` | coast fire, barista fire |
| 5 | `cuando-puedo-jubilarme` | a qué edad me puedo jubilar |
| 6 | `tasa-de-ahorro` | tasa de ahorro, savings rate |
| 7 | `lean-fat-fire` | lean fire vs fat fire |
| 8 | `pension-publica-estimada` | calcular pensión jubilación (España) |
| 9 | `inflacion-poder-adquisitivo` | calculadora inflación |
| 10 | `de-golpe-o-poco-a-poco` | invertir de golpe o poco a poco |
